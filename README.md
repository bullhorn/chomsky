# Chomsky

### Install

Clone/fork this repo and:

```
npm i | yarn
```

### Build

Build once:

```
npm run build | yarn build
```

Watch files and test:

```
npm run watch | yarn watch
```

### Preview

```
npm start | yarn start
```

### Test

Unit tests:

```
npm test | yarn test
```
